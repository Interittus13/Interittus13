### 👋 Hey there

I'm **Arpit Gupta** — a **DevSecOps Engineer** from India who cares about one thing above all else: **shipping fast without shipping risk**.

I design **CI/CD pipelines**, **IaC**, and **cloud-native platforms** where security is baked in from commit to production — not bolted on after an incident.

> *"Security isn't a feature. It's the foundation everything else runs on."*

---

### 🔭 What I'm focused on

```yaml
  building:
    - GitOps microservices pipelines — ArgoCD, Helm, policy-as-code gates
    - Reusable CI/CD templates across GitHub Actions, Jenkins & Azure Pipelines
    - OIDC-based cloud deployments (passwordless, least-privilege by default)

  exploring:
    - AI in DevOps — LLM-assisted runbooks, Copilot-driven pipeline authoring
    - OpenTelemetry for unified traces, metrics & logs on Kubernetes
    - Platform engineering patterns that reduce developer toil

  sharpening:
    - Shift-left security — SAST, SCA, Trivy & secrets scanning on every PR
    - IaC hardening with Terraform, OpenTofu, Checkov & Snyk

  open_to:
    - DevSecOps consulting · pipeline reviews · OSS · speaking at meetups
```
